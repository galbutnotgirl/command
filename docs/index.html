Commands

Every action, zero interruptions.

Built-in bindings use the F-key row. Rebind any action in Settings → Shortcuts, or disable ones you don't need. Dictation and Handoff start unbound.

Text

F8

Comment

Opens a new Claude session with your selection pre-filled. Focus stays in Claude so you can add context before submitting.

⌘F8

Go

Sends your selection to Claude and auto-submits. Focus returns to your previous app immediately after.

⌥F8

Add to Chat

Pastes your selection into the already-open Claude conversation. Great for feeding additional context mid-thread.

Screenshot

F7

Screenshot Comment

Drag to capture any area or press Space for a window. Image lands in a new Claude session; you add a note.

⌘F7

Screenshot Go

Capture → auto-submit. Claude gets the image instantly. Focus returns when it starts responding.

⌥F7

Screenshot Add

Captures and pastes the screenshot into your already-open Claude chat. No new session opened.

Utilities

F6

Clipboard History

Floating picker — type to search, ↑/↓ to select, ⏎ to paste, ⌘⏎ to paste and stay open. Configurable retention.

⌘F6

To-Do

Native popup that routes straight to your intake hook. No Claude session opened — just fast capture.

Dictation — unbound by default

Dictate → Claude

Skill Handoff — unbound by default

Skill Handoff

Routes your selection to a named Claude skill via claude -p in the background. No Claude window opens. Shows a notification when done; logs stay in menu bar → Handoffs.

BACKGROUND

Screenshot Handoff

Capture a region or window → routed to your skill in the background. Same durable submission record as text handoff.

BACKGROUND

Text Handoff

Floating text-entry panel. Type, hit ⌘⏎, and the text goes straight to your skill — no clipboard needed.

How it works

Three steps. Under a second.

Claude Command runs as a background agent. It listens for hotkeys system-wide and handles clipboard capture, context enrichment, focus switching, and the Claude session lifecycle.

✦

Select in any app

Highlight text — in a doc, a browser, a terminal, anywhere on your Mac.

→
⌨

Press your hotkey

F8 to comment. ⌘F8 to go. F7 to capture. Your muscle memory, your keys.

→
◈

Claude has it. You're back.

Selection lands in Claude with source context attached. Go mode auto-submits and returns focus.

Smart Context

Claude knows where it came from.

When you copy from Slack, Gmail, Jira, Google Docs, Gong, or Salesforce, Claude Command automatically tells Claude the source and which MCP to use. No extra typing. Configure your own rules in Settings → Enrichment.

💬

Slack

Finds the exact message via Slack MCP, pulls the channel, thread permalink, author, and surrounding context.

📧

Gmail

Locates the source thread for full email context — not just the snippet you copied.

🟦

Jira / Confluence

Uses the Atlassian MCP to pull the referenced issue or page before Claude responds.

📄

Google Docs & Drive

Reads the full document via gws before acting — never works from a truncated copy.

🎙

Gong

Pulls the related call transcript via the Gong MCP to ground Claude in the full conversation.

☁️

Salesforce

Fetches the related record — account, opportunity, case — before Claude answers about it.

＋

Add your own

Match by bundle ID, URL host (glob), or app name. Write the enrichment text Claude sees. Settings → Enrichment.

Download

Get it for macOS.

One download. Drag to Applications. Grant Accessibility when it asks — that's the whole setup.

Download for macOS .zip · Apple Silicon & Intel · macOS 13+
  1. 1Unzip and drag Claude Command into Applications.
  2. 2First launch: right-click → Open (open-source build, not notarized).
  3. 3Grant Accessibility when prompted. That's the one grant that powers everything.
  4. 4Menu bar ⌘ icon → Settings → Set Up to validate permissions and configure your shortcuts.

Or build from source: ./build-agent.sh && ./install-agent.sh · Updates: Settings → About → Check for Updates
