Import Settings On A New Mac

  1. On old Mac, open Settings -> About -> Import / Export.
  2. Click Export and keep every section checked unless you intentionally want a partial export.
  3. On new Mac, install Command and open Settings -> About -> Import / Export.
  4. Click Import, choose export file, preview available sections, then use Merge for most sections.
